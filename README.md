# 👨‍👩‍👧‍👦 School Parent Management System

Welcome to the **School Parent Management System** – a web-based application built using ASP.NET Core MVC and Web API to manage parent records in a school environment.

---

## 🔧 Project Overview

This system allows school staff to **add**, **edit**, and **view** parent details with clean UI and robust backend validation. The system is designed following modern software engineering practices to ensure maintainability, scalability, and clean separation of concerns.

---

## ✅ Key Features

- Add & edit parent records
- Front-end form validation
- Real-time feedback with success/error messages
- Modular design with reusable components

---

## 📦 Technologies Used

### 🧠 Backend (ASP.NET Core Web API)
- DTOs (Data Transfer Objects) for clean API communication
- Repository Pattern with Interface & Implementation for abstraction and testability
- Model Validation using Data Annotations
- Entity Framework Core for ORM
- Dependency Injection for service management

### 🎨 Frontend (ASP.NET Core MVC)
- Razor Views with strong typing
- Form validation using `asp-validation-for`
- TempData for user-friendly alert messages
- Bootstrap 5 for responsive and elegant UI
- CSS for custom styling

---

## 🛠 Best Practices Followed

- **Separation of Concerns**: Controllers, Services, Repositories, and DTOs are clearly separated.
- **Model Validation**: Ensures only valid data is processed and saved.
- **Exception Handling**: Uses try-catch blocks and displays appropriate feedback.
- **API Design**: Follows REST principles with HTTP status code handling.
- **Reusable UI Components**: Bootstrap-based layout for consistent UI experience.
